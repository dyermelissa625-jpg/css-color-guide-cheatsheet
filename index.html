Format Syntax Alpha? Notes

CSS Custom Properties (Variables)

Step Code Notes
Define :root { --brand: #ff6b6b; } Global scope via :root
Use color: var(--brand); Reference anywhere
Fallback color: var(--brand, red); 2nd arg = fallback value
Override .dark { --brand: #ff9f9f; } Scoped to selector
